Detailed Description

Interval arithmetic.

This class performs interval arithmetic on two intervals by overloading the operators +, -, *, /, +=, -=, *=, /=, <<, >>.

interval interval::operator* ( double const &  obj)

Operator overload for multiplication of an interval and a double.

Parameters
objthe double to multiply this interval by
Returns
the product of the interval and the double as a new interval

This function overloads the * operator to multiply a double by a interval. A temporary interval is created and all permutations of the min and max values are multiplied together. The minimum and maximum values are then found using the interval::find_min and interval::find_max functions and written to the temporary interval. The temporary interval is then returned.
