Detailed Description

Implementation of the interval class.

Author
George Downing
Date
16-12-2022

This file contains the implementation of the interval class. This class performs interval arithmetic on two intervals by overloading the operators +, -, *, /, +=, -=, *=, /=, <<, >>.

DOxygen documentation: https://georgedowning20.github.io/The-Interval-Arithmetic-Project/files.html

interval operator* ( double const &  obj,
interval const &  obj2 
)

This function overloads the * operator to multiply a interval by a double. A temporary interval is created and all permutations of the min and max values are multiplied together. The minimum and maximum values are then found using the interval::find_min and interval::find_max functions and written to the temporary interval. The temporary interval is then returned.
